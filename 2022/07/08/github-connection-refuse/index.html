这与 ssh 的运行机制有关，ssh 会将本机访问过的计算机的 public key 记录在 ~/.ssh/known_hosts 下。当下次访问相同计算机时，若公钥不同则会发出警告，避免受到攻击。这里只需要找到 known_hosts 文件中对应 ip 的 RSA 并删除便可解决。
